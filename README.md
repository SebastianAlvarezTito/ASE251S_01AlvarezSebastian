## ✨ Características Interactivas

| Funcionalidad | Descripción |
|---------------|-------------|
| 🌙 **Modo Oscuro/Claro** | Cambia la apariencia de la tarjeta con un solo clic |
| 💬 **Frases Dinámicas** | Rota entre 5 frases profesionales motivadoras |
| ℹ️ **Info Adicional** | Muestra/oculta datos extras como edad, idiomas y hobbies |
| 🎨 **Diseño Responsivo** | Se adapta perfectamente a dispositivos móviles |
| 🔗 **Enlaces Sociales** | Acceso directo a GitHub y LinkedIn |
